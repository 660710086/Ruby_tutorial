##ซึ่งผมขอทบทวนให้ทุกคนที่เข้ามาอ่านเพื่ออย่างน้อยได้รำลึกถึงสมัยวันวานกันหน่อยนะครับ ซึ่งตามภาพเลยคือความหมายของแต่ละแบบซึ่งในตอนผมยังเด็กๆผมก็มีทริคจำความหมายของพวกนี้ง่ายๆคือ Union = เอาหมด, Intersection = เอาซ้ำ, Difference = เอาตัวไม่ซ้ำ อาจจะตลกไปหน่อยแต่นี่คือทริคการจำของผมครับ 

##ในเมื่อเราได้ทบทวนคร่าวๆไปบ้างแล้วงั้นผมขอเข้าสู่เนื้อหาเลยแล้วกันนะครับ


#1.) Intersection
<pre>
require 'set'

a = Set.new([1, 2, 3])
b = Set.new([3, 4, 5])

puts (a & b).to_a   # Intersection => [3]
puts (a | b).to_a   # Union => [1, 2, 3, 4, 5]
puts (a - b).to_a   # Difference => [1, 2]
</pre>
