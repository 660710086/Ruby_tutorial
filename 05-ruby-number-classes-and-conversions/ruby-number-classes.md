# Ruby Number Classes 
 Ruby Number Classes and conversions มันคืือ แทบทุกอย่างภายใน Ruby มันคือวัตถุ (object) บางทีสิ่งที่น่าประหลาดใจมากที่สุด นั่นก็คือ มันเป็นจำนวนคู่ (even) หรือ ตัวเลขคู่ เป็น วัตถุ (object) ใน Ruby และ ภาษาโปรแกรมอื่นๆ โดยส่วนใหญ่แล้วจะถือว่า ตัวเลข (numbers) เป็น primitive (ชนิดข้อมูลพื้นฐาน) ไม่ใช่ วัตถุ (object) หรือโดยที่พื้นฐานหรือสาระสำคัญเลยจะหมายถึง พวกมันเป็นส่วนประกอบพื้นฐาน (building blocks) จากสิ่งที่วัตถุ(object) สร้างขึ้น และถัดมา แต่ละประเภทตัวเลข (number type) จะมีความหมายที่เกี่ยวข้อง คลาสของตัวเลข(number class) และ สามารถ ถูกจัดการหรือถูกนำไปใช้ โดยใช้ เมธอดที่อยู่ในคลาสนั้น
